# Ecommerce Course Transcripts

Add one `.md` file per lesson. Any filename ending in `.md` is loaded automatically when `/ecommerce` is invoked.

## How to add a lesson

1. Copy your transcript text into a new file: `lesson-02-your-topic.md`
2. Use plain text — no special format required. Headers help but aren't mandatory.
3. The skill reads all `.md` files in this directory in alphabetical order.

## Files in this directory

- `lesson-01-page-frameworks.md` — Homepage, About, FAQ, and Product page checklists + voice rules
